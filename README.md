
# Test para EML Agencia Healthcare



## Installation

Install the project in your pc

```bash
  git clone https://github.com/ElGoel/test-EML
  cd test-EML
  composer install
  php artisan migrate
  php artisan serve

  Database connection: 
    .env file example
    # DB_CONNECTION=sqlite
    # DB_HOST=127.0.0.1
    # DB_PORT=3306
    # DB_DATABASE=laravel
    # DB_USERNAME=root
    # DB_PASSWORD=
```

## API Reference

#### Get all Users

```http
  GET /api/users
```

#### Create a new user

```http
  POST /api/users
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `body` | `raw` | **Required**. all fields |


#### Update User

```http
  GET /api/users/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `body` | `raw` | **Required**. Id of user to update |

```http
  GET /api/users/${id}
```

#### Delete User

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of user to delete |
